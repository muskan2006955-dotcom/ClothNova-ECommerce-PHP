# 👗 Cloth Nova - Online Fashion E-Commerce Platform

**Cloth Nova** is a premium, full-stack E-commerce web application developed for the **IT Vision Competition**. The project is designed to provide a seamless shopping experience for fashion enthusiasts, featuring a modern UI and a robust backend.

---

## 🏆 Competition Highlights
* **Event:** IT Vision Competition
* **Purpose:** To demonstrate scalable web architecture and user-centric design in a retail environment.

---

## 🚀 Key Features

### 🛒 Customer Side
* **Product Catalog:** Browse through various categories with high-quality images.
* **Smart Search & Filtering:** Find products based on variants and categories.
* **Wishlist System:** Save favorite items for later using the integrated `wishlist_toggle` feature.
* **Order Tracking:** Real-time status updates of orders through the `track.php` module.
* **User Profile:** Manage personal details and view order history (`my_orders.php`).
* **Interactive UI:** Smooth transitions, preloaders, and responsive design.

### 🛠 Admin & Backend Features
* **Product Management:** Add, update, and manage product variants and images.
* **Order Management:** View and update order statuses (Pending, Processed, Delivered).
* **Subscriber System:** Manage newsletter subscriptions and customer engagement.
* **Authentication:** Secure login, registration, and password reset functionality.

---

## 🛠️ Tech Stack
* **Backend:** PHP (Core)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, jQuery
* **Plugins:** jQuery Magnify (for product zooming), EZ-Plus.

---

## 📸 Project Structure (Major Modules)
* `index.php`: The main storefront.
* `product-detail.php`: Detailed view with variant selection.
* `tryon.php`: Innovative feature for product preview/try-on.
* `wishlist_show.php`: Personalized wishlist for logged-in users.
* `uploads/`: Dynamic directory for product and profile images.
